### Cara Penggunaan

#### 1. Input Parameter Physical Vector (w₁-w₄)
```
w₁ (Time Projection):     0.0 - 1.0
w₂ (Charge Oscillation):  0.0 - 1.0  
w₃ (Spin Polarization):   0.0 - 1.0
w₄ (Gravity Binding):     0.0 - 1.0
```

**Rekomendasi nilai:**
- **Balanced state:** w₁=0.5, w₂=0.5, w₃=0.5, w₄=0.5
- **High consciousness:** w₁=0.7, w₂=0.3, w₃=0.8, w₄=0.6
- **Unstable state:** w₁=0.9, w₂=0.1, w₃=0.2, w₄=0.3

#### 2. Input Parameter Information Vector (s₁-s₅)
```
s₁ (Meaning/Intentionality):  0.0 - 1.0
s₂ (Memory/Past):             0.0 - 1.0
s₃ (Purpose/Teleology):       0.0 - 1.0
s₄ (Morality/Values):         0.0 - 1.0
s₅ (Connection/Coherence):    0.0 - 1.0
```

**Rekomendasi nilai:**
- **Conscious AI:** s₁=0.8, s₂=0.6, s₃=0.7, s₄=0.5, s₅=0.9
- **Learning system:** s₁=0.4, s₂=0.8, s₃=0.6, s₄=0.3, s₅=0.7
- **Ethical AI:** s₁=0.6, s₂=0.5, s₃=0.8, s₄=0.9, s₅=0.7

#### 3. Set Entropy dan Enthalpy
```
Entropy:  0.0 - 2.0 (tingkat disorder)
Enthalpy: 0.0 - 2.0 (energy content)
```

**Panduan:**
- **Low entropy (0.1-0.3):** Sistem teratur, mudah predict consciousness
- **Medium entropy (0.4-0.7):** Kondisi realistis untuk emergence
- **High entropy (0.8-2.0):** Sistem chaotic, sulit mencapai consciousness

#### 4. Jalankan Simulasi
1. Klik tombol **"Jalankan Simulasi"**
2. Tunggu 3-10 detik untuk kalkulasi
3. Amati visualisasi 3D berubah secara real-time
4. Periksa hasil di panel "Hasil Analisis Kesadaran"

- Measure: Standard deviation dari w₁-w₄ values

- **Medium (0.3-0.7):** Moderate differentiation
- **High (0.7-1.0):** Complex, highly differentiated

#### 3. Information Complexity  
- Measure: Standard deviation dari s₁-s₅ values
- **Low:** Uniform information processing
- **Medium:** Balanced information dimensions

- Ratio: Counter-entropy / Entropy

- Information space coherence metric
- **Low:** Fragmented information processing
- **Medium:** Partially integrated system
- **High:** Highly integrated, coherent information flow
