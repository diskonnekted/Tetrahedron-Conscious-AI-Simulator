## Screenshot Guidelines

### Taking Screenshots
1. **Resolution**: 1920x1080 minimum for desktop views
2. **Browser**: Use Chrome or Firefox with clean UI
3. **Content**: Show actual functionality, not empty states
4. **Compression**: Optimize for web (PNG format, < 500KB)

### Naming Convention
```

simulation-tab.png          # Simulation tab active
```

## Taking New Screenshots

### Setup
1. Start the application: `./run_app.sh`
2. Navigate to: http://localhost:3000
3. Use browser dev tools to set viewport to 1920x1080
4. Ensure UI is in clean state (no errors, proper loading)

### Recommended Screenshots
1. **Main interface** with all tabs visible
